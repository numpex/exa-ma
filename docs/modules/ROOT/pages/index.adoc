== Our Mission

The Exa-MA project is part of France's PEPR NumPEx initiative, dedicated to pushing the frontiers of exascale computing. We develop and deploy:

[.grid.grid-2.gap-2.items-start]
--
[.no-panel]
====
icon:microscope[size=2x] *Advanced Methods*

Novel discretizations, solvers, and algorithms optimized for exascale architectures
====

[.no-panel]
====
icon:brain[size=2x] *AI Integration*

Scientific machine learning for simulation, model reduction, and optimization
====

[.no-panel]
====
icon:code[size=2x] *Open Software*

Production-ready libraries that abstract hardware complexity and enable seamless portability
====

[.no-panel]
====
icon:chart-line[size=2x] *Validated Impact*

Demonstrators, benchmarks, and real-world applications across scientific domains
====
--

[.lead.lead-75]
Exa-MA brings together leading French institutions (CEA, École Polytechnique, Inria, Sorbonne Université, Université de Strasbourg) with a budget of 6.255 M€ over the project duration.

== Work Packages Overview

Exa-MA is organized into seven scientific work packages plus project management:

[discrete]
=== Numerical Methods
[cols="1,4"]
|===
|xref:workpackages/wp1.adoc[WP1] | *Discretization* – Meshing, adaptive refinement, high-order schemes
|xref:workpackages/wp2.adoc[WP2] | *Model Reduction & Scientific ML* – ROMs, PINNs, neural operators
|xref:workpackages/wp3.adoc[WP3] | *Solvers* – Domain decomposition, preconditioners, multiphysics
|===

[discrete]
=== Data & Decision
[cols="1,4"]
|===
|xref:workpackages/wp4.adoc[WP4] | *Inverse Problems & Data Assimilation* – Merging observations with simulations
|xref:workpackages/wp5.adoc[WP5] | *Optimization* – Bayesian optimization, shape and topology optimization
|xref:workpackages/wp6.adoc[WP6] | *Uncertainty Quantification* – Sensitivity analysis, surrogate-based UQ
|===

[discrete]
=== Integration & Coordination
[cols="1,4"]
|===
|xref:workpackages/wp7.adoc[WP7] | *Showroom & Benchmarking* – CI/CD, demonstrators, co-design with ExaDIP
