== Structure

[.grid.grid-2.gap-2]
====
____
icon:cogs[size=2x,role=text-primary] *Methods (WP1-WP6)*

Six scientific work packages develop advanced numerical methods: discretization, model reduction, solvers, inverse problems, optimization, and uncertainty quantification.
____

____
icon:rocket[size=2x,role=text-success] *Integration and Coordination (WP0 & WP7)*

WP0 keeps Exa-MA moving smoothly: governance, coordination, risk tracking, and reporting across all seven technical work packages. WP7 integrates methods into production software, coordinates benchmarking on EuroHPC systems, and ensures co-design with hardware evolution.
____
